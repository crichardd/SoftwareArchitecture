# Utiliser l'application

## Ajouter une tâche

```
python taskmanager.py add "{taskContent}"
```

## Mettre une tâche en terminé

```
python taskmanager.py finish {taskId}
```

## Lister mes tâches

```
python taskmanager.py list
```
